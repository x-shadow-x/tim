招贤纳士

广州 金融产品专员
招聘人数：若干人 +

岗位描述：

  1. 负责市场渠道的开拓，完成渠道的销售任务；
  2. 新渠道开发，渠道商的联络、考评、筛选、淘汰和更新工作；
  3. 行业推广渠道发展趋势分析；负责渠道客户问题的收集、反馈、解决；
  4. 协收集、整理、归纳市场行情、价格，以及新项目、竞争对手、客源等信息资料，提出分析报告，为部门业务人员、领导决策提供参考。

任职要求：

  1. 具备基本的学习能力，有较强的执行能力及沟通能力；
  2. 对销售工作有较高的倾向和热情，想通过销售来改变自己目前的经济状况，改变自己的性格；
  3. 有激情，有梦想；
  4. 有耐心，有韧性；
  5. 思维敏捷，口齿伶俐；
  6. 有电商经验者优先；
  7. 有招商经验者优先；
  8. 懂股票知识者优先；
广州 文案策划
招聘人数：若干人 +

岗位描述：

  1. 负责微信、微博等自媒体的财经、金融类文章编辑工作；
  2. 对自媒体文章进行选题、撰稿、筛选、推送、维护；
  3. 关注财经热点，负责线上营销专题策划、实施、总结！
  4. 对用户增长，内容阅读量负责，做好用户数据的统计、分析；
  5. 与相关部门沟通协作，利用新媒体渠道完成内容推广、引流的工作；
  6. 协助上级完成其他工作。

任职要求：

  1. 大专及以上学历，1年以上财经/金融类编辑、文案经验，具备原创能力；
  2. 1年以上微信或者微博自媒体运营经验；
  3. 对金融有一定了解，善于发现财经热点，有创新精神；
  4. 面试需携带个人作品。
广州 企划经理
招聘人数：若干人 +

岗位描述：

  1. 研究行业发展动态，为公司及品牌的发展提供战略规划支持；
  2. 根据市场中心的业务发展需求，组织企划部进行金融产品营销策划、促销活动、宣传物料设计制作及印刷（折页、海报、画册、易企秀等）、网络推广、数据收集分析；
  3. 研究公司文化、品牌理念及VI系统，进行宣传、维护、升级。定期策划、组织员工文化活动；
  4. 负责公司网站的设计、建设、完善与维护；
  5. 负责与各金融机构、广告公司、网络科技公司等机构联络、接洽，建立良好的合作关系；
  6. 上司交待的其它事项。

任职要求：

  1. 本科学历，3年以上相关岗位从业经验，具备较强的品牌策划、资源整合与营销能力。
  2. 擅长广告、文案、设计、网络推广、微信公众号运营、视频拍摄制作等其中一种或多种技能、经验。
  3. 工作主动积极，具有高度责任心，团队管理与协作能力出色。
广州 财务主管
招聘人数：若干人 +

岗位描述：

  1. 对接财务系统需求，定期统计相关报表，随时跟踪产品运营情况；
  2. 根据公司经营计划，组织年度综合财务计划和控制标准；
  3. 审核财务单据，整理档案，管理发票，审核记账凭证、核对调整账目、预算分析、控制日常费用、管理固定资产；
  4. 协助制订集团财务预算及税务筹划方案，并跟进落实；
  5. 上级领导安排的其他临时性工作。

任职要求：

  1. 至少5年以上财会类岗位工作经验，本科以上学历，有证券、基金、金融行业经验者优先；
  2. 熟练掌握Excel操作，熟悉金蝶、ERP系统操作，具备一定的账务处理及核算能力；
  3. 有沟通协调能力强、抗压能力强、具备团队合作精神。
广州 财务会计
招聘人数：若干人 +

岗位描述：

  1. 审核原始凭证，编制记账凭证，登记总分类帐和明细帐。
  2. 核查现金、银行日记帐及往来帐项，并及时催缴各项应收款；
  3. 负责固定资产的财务管理，按月计提固定资产折旧；
  4. 负责本部门预算报表的编制；
  5. 负责公司税金的计算、抄税、申报和解缴；
  6. 负责发票的购入、保管、开具、及缴销；
  7. 网上银行转帐的审核及及员工工资的核算；
  8. 负责编制各项报表、汇总表及进度表，包括：月报、季报、年报；
  9. 负责配合财务经理、事务所审计人员完成财务审计；
  10. 如实反映公司经营状况，提出改进建议和措施；
  11. 负责每年公司的银行、税务的年检；
  12. 负责凭证帐本的打印、归档，财务档案、税务文书资料保管；
  13. 完成财务经理安排的其他工作

备注：有投资公司经验者优先

任职要求：

  1. 专科及以上学历；
  2. 会计或金融专业者优先；
  3. 有会计从业资格证；
  4. 能独立、熟练的处理全盘帐务及纳税申报。
广州 董事长助理
招聘人数：若干人 +

岗位描述：

  1. 负责董事长的日程安排，适时提醒（包括参加会议、约见客人、处理事务、批示文件等）；
  2. 负责董事长出差行程的安排（包括护照签证、预订机票、送机接机、住宿、餐饮、会见等一系列活动，准备好出差需要的相关文件、资料）；
  3. 负责董事长重要客人的接待和安排工作（包括茶室、水果、点心、餐饮、住宿等准备）；
  4. 根据董事长的要求起草文件、发言稿及相关管理制度，熟悉公司运营，对业务产品及销售团队管理能起到协办和监督；
  5. 负责文件、资料的流转及流转中的工作（根据紧急、重要程度对来文进行分类，对紧急的文件及时安排董事长批阅，协助董事长高效处理文件。根据董事长的批示，对需要继续流转的文件及时传递；对处理完毕的文件及时反馈和归档，做好流转中的工作；
  6. 负责督察督办工作（根据董事长书面批示和口头交代的事项，按时限做好督察督办工作，及时跟进、落实和反馈）；
  7. 负责会议管理（协助大型会议、活动的筹办；对召集的会议负责组织、协调、跟踪工作，做好会议记录，并跟踪督办会议形成的决议内容）；
  8. 协助董事长维护良好的外部客户关系，及时更新董事长名片管理系统，协助公司相关部门做好董事长的重大社交、公益活动和社会职务等活动的联络、协调工作；
  9. 负责董事长个人物品的收集、整理、归档工作（保管总经理的个人证照，保险柜的钥匙，做好使用、借阅登记）；
  10. 负责董事长各项公务费用的报销工作；
  11. 负责董事长关活动贵重物资的采购和管理工作；
  12. 维护董事长办公室环境的整洁、有序、安全。

任职要求：

  1. 3年以上行政、人事、文秘工作经验，有1年以上金融或投资公司的经验；
  2. 优秀的语言表达能力、写作能力和沟通能力，良好的公关能力和时间管理能力，思路清晰，认真细心，执行力强，熟悉企业经营管理各个层面的工作，熟悉使用办公软件，较强的承受工作压力能力；
  3. 身高160CM以上，相貌较好，谈吐斯文；
  4. 本科学历，文秘或行政管理或金融专业；
  5. 英语六级或专八；
  6. 熟悉国内外金融咨询者优先；
  7. 投递简历时请附上个人近期生活照一张。
